ZTApi

This object provides an API for application user

  • onSavedRequestSent

    Event to subscribe notifiing about saved request was sent

  • onActivityStopRequestSent

    Event to subscribe notifiing about ‘stop’ request was sent

  • onSocketDataReceived

    Event to subscribe notifiing about data received in webscoket

  • onSocketTextReceived

    Event to subscribe notifiing about message received in webscoket

  • appAttributes

    Application attributes

  • linkUser(userId:attributes:completion:)

    Associate an application user with unique ID and record traits about it. This method should be called after each signup / signin.

    Get the list of shoes the logged in user has scanned

    Get user’s shoe by code
